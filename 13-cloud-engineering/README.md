## 학습 로드맵

### 1단계: 기초 (1-2개월)
```
- [ ] Linux 명령어 숙달 (top, ps, grep, awk)
- [ ] Docker로 간단한 앱 컨테이너화
- [ ] AWS EC2에 수동 배포 경험
```

### 2단계: 실무 입문 (3-4개월)
```
- [ ] K8s 로컬 환경 구축 (Minikube)
- [ ] GitHub Actions로 CI 파이프라인 구축
- [ ] Prometheus + Grafana 설치
```

### 3단계: 포트폴리오 (5-6개월)
```
- [ ] 3-tier 웹 앱을 AWS + K8s로 배포
  - VPC 네트워크 설계
  - EKS 클러스터 구축
  - ArgoCD로 GitOps 배포
  - Prometheus 모니터링
```
