# 백엔드 엔지니어링 (Backend Engineering)

> Java와 Spring을 중심으로 한 백엔드 개발자 필수 지식

## 학습 목표

- [ ] Java의 핵심 개념과 최신 기능을 이해한다
- [ ] Spring Framework의 동작 원리를 파악한다
- [ ] 데이터베이스와 JPA를 활용할 수 있다
- [ ] 인증/인가 메커니즘을 구현할 수 있다

---

## 하위 폴더 구조

```
02-backend-engineering/
├── java-fundamentals/     # Java 기초 및 심화
├── spring-framework/      # Spring Core, Boot, MVC
├── database/              # JPA, 트랜잭션, 인덱싱
└── authentication/        # Session, JWT, OAuth
```

---

## 학습 순서 (권장)

1. **Java 기초** - OOP, SOLID, 메모리 모델
2. **Java 심화** - GC, Virtual Threads
3. **Spring Core** - IoC/DI, AOP
4. **Spring Boot** - Auto Configuration
5. **Database** - JPA, 트랜잭션, 최적화
6. **인증** - Session, JWT, OAuth
