# 컴퓨터 구조 (Computer Architecture)

> 컴퓨터 하드웨어의 동작 원리와 설계 개념

## 학습 목표

- [ ] CPU의 구성요소와 명령어 실행 과정을 이해한다
- [ ] 캐시 메모리의 필요성과 동작 방식을 안다
- [ ] 수 표현 방식(고정/부동 소수점)의 차이를 이해한다

## 학습 순서 (권장)

1. **CPU 구조** - ALU, 제어장치, 레지스터, 명령어 사이클
2. **캐시 메모리** - 계층 구조, 지역성 원리
3. **수 표현** - 고정 소수점, 부동 소수점
4. **오류 검출** - 패리티 비트, 해밍 코드
5. **프로세서 설계** - ARM vs x86 (RISC vs CISC)
