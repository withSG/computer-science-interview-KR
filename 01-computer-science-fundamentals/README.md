## 학습 순서 (권장)

1. **운영체제** - 프로세스, 스레드, 메모리 관리
2. **네트워크** - OSI 모델, TCP/IP, HTTP
3. **자료구조** - 배열, 리스트, 트리, 해시
4. **알고리즘** - 시간복잡도, 정렬, 탐색

## 면접 빈출 주제

| 주제 | 빈출도 | 핵심 키워드 |
|------|--------|-----------|
| 프로세스 vs 스레드 | ⭐⭐⭐ | 메모리 공유, 컨텍스트 스위칭 |
| 데드락 | ⭐⭐⭐ | 4가지 조건, 회피/탐지 |
| HTTP vs HTTPS | ⭐⭐⭐ | SSL/TLS, 암호화 |
| TCP vs UDP | ⭐⭐ | 신뢰성, 연결 지향 |
| 해시 테이블 | ⭐⭐⭐ | 충돌 해결, O(1) |
| 시간 복잡도 | ⭐⭐⭐ | Big-O 표기법 |
