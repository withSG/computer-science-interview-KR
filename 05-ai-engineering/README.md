# AI 엔지니어링 (AI Engineering)

> RAG, 벡터 DB, LLM 통합의 핵심 개념

## 학습 목표

- [ ] RAG 파이프라인의 전체 구조를 이해한다
- [ ] 벡터 데이터베이스의 원리를 안다
- [ ] LLM 통합과 프롬프트 엔지니어링을 할 수 있다
- [ ] 환각(Hallucination) 완화 기법을 안다

## 학습 순서 권장

```
1. LLM 기초 (모델 이해, 프롬프트 엔지니어링)
       ↓
2. 임베딩과 벡터 DB (유사도 검색)
       ↓
3. RAG 파이프라인 (청킹, 검색, 생성)
```
