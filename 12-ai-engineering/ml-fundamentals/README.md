# ML 기초 (Machine Learning Fundamentals)

> 머신러닝/딥러닝 직무 면접의 핵심 기초 개념

## 학습 목표

- [ ] 과적합/과소적합과 완화 방법을 설명할 수 있다
- [ ] L1/L2 정규화의 차이를 안다
- [ ] 전이학습(Transfer Learning)의 개념과 활용을 안다
- [ ] 분류/회귀 평가지표를 구분하고 선택할 수 있다
- [ ] 편향-분산 트레이드오프를 이해한다
- [ ] 설명 가능한 AI(XAI)의 필요성과 기법을 안다
