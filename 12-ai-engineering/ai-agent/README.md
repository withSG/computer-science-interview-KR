# AI Agent / Workflow (LangGraph)

> LLM이 스스로 추론하고 도구를 사용하는 에이전트와 워크플로우 설계

## 학습 목표

- [ ] AI Agent의 개념과 동작 방식(추론-행동 루프)을 이해한다
- [ ] LangGraph의 핵심 개념(State, Node, Edge)을 설명할 수 있다
- [ ] 조건부 분기와 Human-in-the-Loop를 적용할 수 있다
- [ ] 무한 루프 방지 등 안정적인 에이전트 운영 방법을 안다
- [ ] 프롬프트 엔지니어링과 플로우 엔지니어링의 차이를 안다
- [ ] AI 하네스(Agent Harness)의 역할을 이해한다
- [ ] BMAD Method 등 에이전트 기반 개발 방법론을 안다
