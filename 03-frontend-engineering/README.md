# 프론트엔드 엔지니어링 (Frontend Engineering)

> 브라우저부터 React, Next.js, TypeScript, 아키텍처까지 프론트엔드 핵심 개념

## 학습 목표

- [ ] 브라우저 렌더링 과정을 설명할 수 있다
- [ ] JavaScript 실행 컨텍스트와 이벤트 루프를 이해한다
- [ ] React의 가상 DOM과 Fiber 아키텍처를 안다
- [ ] Next.js의 렌더링 전략을 비교할 수 있다
- [ ] HTML/CSS 레이아웃과 반응형 웹을 구현할 수 있다
- [ ] TypeScript의 타입 시스템을 이해한다
- [ ] Webpack과 Babel의 역할을 설명할 수 있다
- [ ] 프론트엔드 아키텍처 설계 원칙을 안다

## 학습 순서 권장

```
1. HTML/CSS 기초 (시멘틱, 반응형, Flexbox/Grid)
       ↓
2. 브라우저 기초 (CRP, DOM/CSSOM)
       ↓
3. JavaScript 심화 (실행 컨텍스트, 이벤트 루프)
       ↓
4. TypeScript (타입 시스템, 제네릭)
       ↓
5. 빌드 도구 (Webpack, Babel)
       ↓
6. React 아키텍처 (가상 DOM, Fiber, Hooks)
       ↓
7. Next.js 렌더링 (SSR, RSC)
       ↓
8. 프론트엔드 아키텍처 (상태 관리, 컴포넌트 설계)
```
