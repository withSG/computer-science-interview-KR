# 프론트엔드 엔지니어링 (Frontend Engineering)

> 브라우저부터 React, Next.js까지 프론트엔드 핵심 개념

## 학습 목표

- [ ] 브라우저 렌더링 과정을 설명할 수 있다
- [ ] JavaScript 실행 컨텍스트와 이벤트 루프를 이해한다
- [ ] React의 가상 DOM과 Fiber 아키텍처를 안다
- [ ] Next.js의 렌더링 전략을 비교할 수 있다
