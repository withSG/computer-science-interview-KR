## 면접 준비 팁

### 트러블슈팅 질문 답변 구조

```
1. 상황 설명 (어떤 문제였는지)
2. 진단 과정 (어떻게 원인을 찾았는지)
3. 해결 방법 (어떤 조치를 취했는지)
4. 결과 및 학습 (이후 개선점)
```

### 아키텍처 설계 질문 접근법

```
1. 요구사항 명확화 (질문하기)
2. 대략적인 설계 (High-level)
3. 핵심 컴포넌트 상세화
4. 확장성/가용성 고려
5. 트레이드오프 설명
```

### STAR 기법

```
S - Situation (상황)
T - Task (과제)
A - Action (행동)
R - Result (결과)
```
